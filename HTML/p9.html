HTML LIST

There are 3 types of list

1) Ordered or Number list (Using OL Tag)

2) Unordered or Bullet list (Using UL Tag)

3) Description List (Using DL Tag)

Nested list Example..

  1. Karnataka
    • Bengaluru
  2. Andhra Pradesh
    • Amaravati
  3. Tamil Nadu
    • Chennai
  4. Hariyana
    • Chandigarh
  5. Gujarat
    • Gandhi Nagar
